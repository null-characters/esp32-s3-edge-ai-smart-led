# prj-v3 代码审查计划总览

## 项目概述
prj-v3 是 ESP32-S3 Edge AI Smart LED 项目的语音交互升级版本，引入了多模态 AI 推理架构。

## 审查重点总览

### 安全性审查（所有模块）
- 内存安全（缓冲区溢出、空指针）
- 并发安全（竞态条件、死锁）
- 输入验证（边界检查）

### 性能审查（Plan 02, 03, 04）
- 实时性要求（音频处理 < 10ms）
- 内存占用（PSRAM 使用）
- CPU 利用率

### ESP-IDF 合规性（所有 C 文件）
- FreeRTOS API 使用规范
- ESP-IDF 驱动接口规范
- 日志系统规范
