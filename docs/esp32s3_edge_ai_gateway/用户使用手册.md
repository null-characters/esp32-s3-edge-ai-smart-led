# ESP32-S3 Edge AI Smart LED 用户使用手册

> 多模态 AI 智能照明系统 - 语音交互版本 (v3)

---

## 产品概述

### 功能特性

| 功能 | 说明 |
|------|------|
| **语音控制** | 唤醒词"小白小白" + 50+ 命令词，本地离线识别 |
| **自动控制** | 雷达人体检测 + AI 场景识别 |
| **双轨融合** | 语音优先，TTL 租约自动恢复机制 |
| **亮度调节** | 0-100% 无级调光 + 主灯/辅助灯独立控制 |
| **色温调节** | 2700K-6500K 智能调色 |
| **场景预设** | 专注、会议、演示、休息、明亮、夜间 6 种模式 |

### 技术规格

| 项目 | 规格 |
|------|------|
| 主控芯片 | ESP32-S3-N16R8 (16MB Flash + 8MB PSRAM) |
| 语音引擎 | ESP-SR WakeNet (xiaobaitong) + MultiNet |
| AI 推理 | TensorFlow Lite Micro (INT8, ~21KB) |
| 响应延迟 | < 1 秒 (端到端) |
| 工作电压 | 5V DC |
| 功耗 | < 5W (待机) / < 20W (满载) |

## 硬件组装

### 必需组件

| 组件 | 型号 | 数量 |
|------|------|------|
| 开发板 | ESP32-S3-N16R8 | 1 |
| 麦克风模块 | INMP441 | 1 |
| 扬声器模块 | MAX98357A + 3W喇叭 | 1 |
| 雷达模块 | LD2410 | 1 |
| LED 调光模块 | PWM 驱动板 (亮度+色温 或 RGB) | 1 |

### 接线说明

> ⚠️ **注意**: DAC (扬声器) 和 UART (雷达) 共用 GPIO 16-18，需要根据实际配置调整

#### 麦克风 (INMP441) - I2S0

| 麦克风引脚 | ESP32-S3 引脚 |
|-----------|---------------|
| VDD | 3.3V |
| GND | GND |
| CLK (SCK) | GPIO 14 |
| WS (LRCK) | GPIO 12 |
| SD (DATA) | GPIO 13 |

#### 扬声器 (MAX98357A) - I2S1

| 扬声器引脚 | ESP32-S3 引脚 |
|-----------|---------------|
| VIN | 5V |
| GND | GND |
| BCLK | GPIO 16 |
| LRC (WS) | GPIO 17 |
| DIN | GPIO 18 |

#### 雷达 (LD2410) - UART1

> ⚠️ **冲突警告**: 雷达 UART 与扬声器 DAC 共用 GPIO 17/18
> 解决方案: 使用不同的 GPIO 或仅启用其中一个功能

| 雷达引脚 | ESP32-S3 引脚 (默认) | 建议修改 |
|---------|---------------|---------|
| VCC | 5V | 5V |
| GND | GND | GND |
| TX | GPIO 18 | GPIO 19 或 GPIO 20 |
| RX | GPIO 17 | GPIO 21 或 GPIO 22 |

#### LED PWM

**亮度+色温模式 (双通道):**

| LED 引脚 | ESP32-S3 引脚 |
|---------|---------------|
| 亮度控制 | GPIO 4 |
| 色温控制 | GPIO 5 |

**RGB 模式 (三通道):**

| LED 引脚 | ESP32-S3 引脚 |
|---------|---------------|
| R (红) | GPIO 6 |
| G (绿) | GPIO 7 |
| B (蓝) | GPIO 15 |

## 快速开始

### 第一步：烧录固件

```bash
# 连接开发板到电脑 USB
cd prj-v3

# 编译固件
idf.py build

# 烧录固件
idf.py -p /dev/ttyUSB0 flash

# 或使用一键烧录脚本
./scripts/flash.sh
```

### 第二步：首次配置

设备启动后会自动进入配网模式：

1. 手机搜索 Wi-Fi 热点：`SmartLED-XXXX`
2. 连接热点，浏览器访问：`192.168.4.1`
3. 选择家庭 Wi-Fi，输入密码
4. 设备自动连接，语音提示"网络连接成功"

### 第三步：语音唤醒

对着麦克风说出唤醒词：

> **"小白小白"**

设备播放提示音，进入命令等待状态（约 5 秒）。

## 自动控制模式

### 工作原理

```
雷达检测 → AI 分析 → 自动调光
    ↓
  有人/无人
    ↓
  静止/运动
    ↓
  场景识别
```

### 场景自动切换

| 人体状态 | 自动场景 | 亮度 |
|---------|---------|------|
| 无人 | 关灯 | 0% |
| 有人静止 | 专注模式 | 60% |
| 有人移动 | 会议模式 | 70% |

### 环境变化响应

- 人离开 > 30 秒 → 灯光渐暗
- 人离开 > 10 分钟 → 自动关灯
- 光线变化 → 自动调节亮度

## 双轨融合机制

### 语音优先原则

```
语音命令 → 立即生效 → 锁定 2 小时 (TTL)
                         ↓
                    自动控制暂停
                         ↓
                    TTL 过期后恢复
```

### 自动恢复条件

| 条件 | 说明 |
|------|------|
| TTL 过期 | 语音锁定 2 小时后自动释放 |
| 环境变化 | 人离开 > 10 分钟自动释放 |
| 手动恢复 | "恢复自动"命令立即释放 |
| 设备重启 | 恢复之前状态 (NVS 持久化) |

## 故障排除

### 常见问题

| 问题 | 可能原因 | 解决方法 |
|------|---------|---------|
| 无法唤醒 | 麦克风未连接 | 检查 INMP441 接线 (GPIO 12/13/14) |
| 无声音输出 | DAC 未连接 | 检查 MAX98357A 接线 (GPIO 16/17/18) |
| 雷达不工作 | UART 冲突 | 修改雷达 GPIO避开 17/18 |
| Wi-Fi 连接失败 | 密码错误 | 重新配网 |

### 恢复出厂设置

```bash
./scripts/erase_flash.sh
```

1. **电源安全**：使用合规 5V 电源适配器
2. **防水防潮**：设备不防水，避免潮湿环境
3. **散热**：确保设备通风良好
4. **GPIO 冲突**：注意 DAC 和 UART 共用 GPIO 17/18
