# ESP32-S3 Edge AI Smart LED 用户使用手册

> 多模态 AI 智能照明系统 - 语音交互版本 (v3)

---

## 产品概述

### 功能特性

| 功能 | 说明 |
|------|------|
| **语音控制** | 唤醒词"小白小白" + 50+ 命令词，本地离线识别 |
| **自动控制** | 雷达人体检测 + AI 场景识别 |
| **双轨融合** | 语音优先，TTL 租约自动恢复机制 |
| **状态指示灯** | WS2812 RGB LED 实时显示系统状态 |
| **亮度调节** | 0-100% 无级调光 + 主灯/辅助灯独立控制 |
| **色温调节** | 2700K-6500K 智能调色 |
| **场景预设** | 专注、会议、演示、休息、明亮、夜间 6 种模式 |

### 技术规格

| 项目 | 规格 |
|------|------|
| 主控芯片 | ESP32-S3-N16R8 (16MB Flash + 8MB PSRAM) |
| 语音引擎 | ESP-SR WakeNet + MultiNet |
| AI 推理 | TensorFlow Lite Micro (INT8) |
| 状态指示灯 | WS2812 RGB LED (GPIO48) |
| 响应延迟 | < 1 秒 (端到端) |
| 工作电压 | 5V DC |

## 状态指示灯

开发板板载 WS2812 RGB LED (GPIO48)，实时显示系统状态。

### WiFi 连接状态

| 状态 | 颜色 | 效果 | 说明 |
|------|------|------|------|
| 未连接 | 🔴 红色 | 慢闪 (1s) | WiFi 断开 |
| 连接中 | 🟡 黄色 | 快闪 (250ms) | 正在连接 |
| 已连接 | 🟢 绿色 | 常亮 | 连接成功 |
| 连接失败 | 🔴 红色 | 常亮 | 密码错误 |

### 系统运行模式

| 模式 | 颜色 | 效果 |
|------|------|------|
| 自动模式 | 🔵 蓝色 | 常亮 |
| 手动模式 | 🔵 青色 | 常亮 |
| 语音模式 | 🟣 紫色 | 常亮 |
| 睡眠模式 | ⚫ 熄灭 | - |

### 语音交互状态

| 状态 | 颜色 | 效果 |
|------|------|------|
| 监听中 | ⚪ 白色 | 呼吸 (2s) |
| 处理中 | ⚪ 白色 | 快闪 (200ms) |
| 命令成功 | 🟢 绿色 | 快闪 3 次 |
| 命令失败 | 🔴 红色 | 快闪 3 次 |

### 特殊状态

| 状态 | 颜色 | 效果 |
|------|------|------|
| 系统启动 | 🌈 彩虹 | 渐变 (3s) |
| 系统错误 | 🔴 红色 | 快闪 |
| 固件更新 | 🟡 黄色 | 呼吸 |

|------|----------|
| CLK | GPIO 14 |
| WS | GPIO 12 |
| SD | GPIO 13 |

#### 扬声器 (MAX98357A) - I2S1

| 引脚 | ESP32-S3 |
|------|----------|
| BCLK | GPIO 16 |
| LRC | GPIO 17 |
| DIN | GPIO 18 |

#### 雷达 (LD2410) - UART1

| 引脚 | ESP32-S3 |
|------|----------|
| TX | GPIO 19 |
| RX | GPIO 21 |

#### LED PWM

| 引脚 | ESP32-S3 |
|------|----------|
| 亮度 | GPIO 4 |
| 色温 | GPIO 5 |

## 快速开始

### 1. 烧录固件

```bash
cd prj-v3
idf.py build
idf.py -p /dev/ttyUSB0 flash
```

### 2. 观察启动状态

1. **🌈 彩虹渐变** - 系统启动中
2. **🟡 黄色快闪** - WiFi 连接中
3. **🟢 绿色常亮** - WiFi 连接成功
4. **🔵 蓝色常亮** - 系统就绪

### 3. 语音唤醒

说出唤醒词：**"小白小白"**

指示灯变为 **⚪ 白色呼吸**，等待命令。

## 自动控制模式

### 场景自动切换

| 人体状态 | 自动场景 | 亮度 |
|---------|---------|------|
| 无人 | 关灯 | 0% |
| 有人静止 | 专注模式 | 60% |
| 有人移动 | 会议模式 | 70% |

### 环境响应

- 人离开 > 30 秒 → 灯光渐暗
- 人离开 > 10 分钟 → 自动关灯

---

## 双轨融合机制

```
语音命令 → 立即生效 → 锁定 2 小时 (TTL)
                         ↓
                    自动控制暂停
                         ↓
                    TTL 过期后恢复
```

| 恢复条件 | 说明 |
|------|------|
| TTL 过期 | 语音锁定 2 小时后释放 |
| 环境变化 | 人离开 > 10 分钟 |
| 手动恢复 | "恢复自动"命令 |
| 设备重启 | NVS 持久化恢复 |
