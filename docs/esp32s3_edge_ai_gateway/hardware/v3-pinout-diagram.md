# ESP32-S3 N16R8 v3 硬件连接框图

> 基于 ESP32-S3-NxRx 开发板（兼容 ESP32-S3-WROOM-1 模组）的智能照明网关硬件配置

## 一、开发板规格

| 特性 | 规格 |
|------|------|
| 型号 | ESP32-S3-NxRx 开发板 (兼容 ESP32-S3-WROOM-1) |
| 模组 | ESP32-S3-WROOM-1-N16R8 |
| 内核 | Xtensa LX7 双核，最高 240MHz |
| 内部 SRAM | 512KB |
| 外部 Flash | 16MB (Quad SPI) |
| 外部 PSRAM | 8MB (Octal SPI) |
| Wi-Fi | 802.11 b/g/n，2.4GHz |
| 蓝牙 | BLE 5.0 |
| USB | USB OTG (Full-speed) |
| GPIO | GPIO0-GPIO47 (共48个) |
| 板载RGB | WS2812 (GPIO48/特殊引脚) |
| 天线 | PCB天线 (可选外接天线座) |

### 开发板特殊说明

- **5Vin引脚**：输入5V给开发板供电，**不是输出5V**
- **USB-OTG跳线**：短接背面焊盘后可对外部USB从设备供电
- **板载WS2812**：用于状态指示，与外部照明LED分开
- **Arduino IDE开发板选择**：ESP32S3 Dev Module

## 三、GPIO 引脚分配表

### 3.1 音频接口

| GPIO | 功能 | 方向 | 外设 | 说明 |
|------|------|------|------|------|
| **12** | I2S_WS | 输出 | INMP441 麦克风 | 字选择信号 |
| **13** | I2S_SD | 输入 | INMP441 麦克风 | 音频数据输入 |
| **14** | I2S_CLK | 输出 | INMP441 麦克风 | 位时钟信号 |
| **16** | I2S_BCLK | 输出 | MAX98357A DAC | 位时钟输出 |
| **17** | I2S_WS | 输出 | MAX98357A DAC | 字选择输出 |
| **18** | I2S_DIN | 输出 | MAX98357A DAC | 音频数据输出 |

### 3.2 LED PWM 接口 (外部照明)

| GPIO | 功能 | 通道 | 说明 |
|------|------|------|------|
| **4** | LEDC_CH0 | PWM | 亮度控制 (0-100%) |
| **5** | LEDC_CH1 | PWM | 色温控制 (2700K-6500K) |
| **6** | LEDC_CH2 | PWM | RGB - 红色通道 |
| **7** | LEDC_CH3 | PWM | RGB - 绿色通道 |
| **15** | LEDC_CH4 | PWM | RGB - 蓝色通道 |

### 3.3 板载状态指示

| GPIO | 功能 | 类型 | 说明 |
|------|------|------|------|
| **48** | WS2812_DATA | 单线协议 | 板载RGB LED状态指示 |

> ⚠️ **注意**：GPIO48 用于板载 WS2812 RGB LED（状态指示），与外部 LED PWM 照明控制分开。

### 3.4 雷达通信接口

| GPIO | 功能 | 方向 | 外设 | 说明 |
|------|------|------|------|------|
| **8** | UART_TX | 输出 | LD2410 雷达 | 数据发送 |
| **9** | UART_RX | 输入 | LD2410 雷达 | 数据接收 |

### 3.4 预留/未使用 GPIO

| GPIO 范围 | 状态 | 备注 |
|-----------|------|------|
| GPIO0-3 | 预留 | 启动引脚/Flash |
| GPIO10-11 | 可用 | 预留扩展 |
| GPIO19-21 | 可用 | 预留扩展 |
| GPIO35-48 | 可用 | 预留扩展 |

### 电源引脚说明

| 引脚 | 方向 | 说明 |
|------|------|------|
| **USB-C** | 输入 | 5V/2A 主供电接口 |
| **5Vin** | **输入** | 外部5V输入供电（**注意：不是输出**） |
| **3.3V** | 输出 | LDO稳压后3.3V输出（最大800mA） |
| **GND** | - | 公共地 |

### USB-OTG 功能

开发板背面有 USB-OTG 跳线焊盘：
- **默认断开**：USB-C 仅用于供电和调试
- **短接后**：ESP32-S3 可作为 USB Host，对外部 USB 从设备供电

## 六、内存分配策略

| 内存区域 | 大小 | 用途 |
|----------|------|------|
| 内部 SRAM | ~200KB | FreeRTOS任务栈、中断向量、关键DMA缓冲 |
| PSRAM | ~6MB | TFLM Tensor Arena、ESP-SR AEC缓冲、音频缓冲 |
| PSRAM | ~2MB | cJSON解析、HTTP缓冲、历史数据存储 |
