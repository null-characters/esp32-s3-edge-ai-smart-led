## IT1: 端到端功能测试

### IT1.1 语音控制链路测试

**测试目标**: 验证唤醒→命令→执行的完整链路

| 用例ID | 测试场景 | 输入 | 预期输出 | 验收标准 |
|--------|---------|------|---------|---------|
| IT1.1.1 | 唤醒测试 | "小白小白" | 唤醒提示音 + LED 反馈 | 识别率 > 95% |
| IT1.1.2 | 开灯命令 | "打开电灯" | LED 开启 + TTS 确认 | 延迟 < 1s |
| IT1.1.3 | 关灯命令 | "关闭电灯" | LED 关闭 + TTS 确认 | 延迟 < 1s |
| IT1.1.4 | 亮度调节 | "亮度调到50%" | LED 亮度 50% | 精度 ±5% |
| IT1.1.5 | 色温调节 | "色温调到暖光" | LED 色温 3000K | 精度 ±200K |
| IT1.1.6 | 状态查询 | "电灯状态" | TTS 播报当前状态 | 延迟 < 1s |
| IT1.1.7 | 模式切换 | "切换到阅读模式" | 预设场景应用 | 延迟 < 1s |

**测试方法**:
1. 在安静环境 (噪声 < 40dB) 下进行
2. 距离麦克风 1-3 米
3. 每个命令重复 50 次
4. 记录识别率和延迟

### IT1.2 自动控制链路测试

**测试目标**: 验证雷达→AI推理→自动调光的链路

| 用例ID | 测试场景 | 输入 | 预期输出 | 验收标准 |
|--------|---------|------|---------|---------|
| IT1.2.1 | 无人场景 | 雷达无检测 | LED 关闭 | 延迟 < 30s |
| IT1.2.2 | 有人静止 | 雷达检测静止 | 阅读场景 | 亮度 80% |
| IT1.2.3 | 有人移动 | 雷达检测移动 | 活动场景 | 亮度 60% |
| IT1.2.4 | 环境变化 | 光线变暗 | 自动调亮 | 响应 < 5s |
| IT1.2.5 | 声音触发 | 检测到特定声音 | 场景切换 | 分类正确 |

**测试方法**:
1. 模拟不同人体状态 (静止/移动/离开)
2. 验证 AI 推理结果
3. 检查控制命令正确性

### IT1.3 双轨融合测试

**测试目标**: 验证语音干预和自动恢复机制

| 用例ID | 测试场景 | 初始状态 | 操作 | 预期结果 |
|--------|---------|---------|------|---------|
| IT1.3.1 | 语音覆盖 | 自动模式 | 语音命令 | 立即切换到语音控制 |
| IT1.3.2 | TTL 锁定 | 语音控制后 | 等待 | 保持语音模式 2 小时 |
| IT1.3.3 | TTL 释放 | 语音锁定中 | 2 小时后 | 自动恢复自动模式 |
| IT1.3.4 | 手动恢复 | 语音锁定中 | "恢复自动" | 立即释放锁定 |
| IT1.3.5 | 环境释放 | 语音锁定中 | 人离开 10 分钟 | 自动释放锁定 |
| IT1.3.6 | 重启恢复 | 语音锁定中 | 重启设备 | 恢复之前状态 |

**测试方法**:
1. 使用 NVS 存储状态
2. 模拟时间流逝 (加速测试)
3. 验证状态机转换

## IT2: 性能测试

### IT2.1 内存占用测试

| 测试项 | 目标值 | 测试方法 |
|--------|--------|---------|
| 静态内存 | < 200KB | 编译时分析 |
| 动态内存峰值 | < 800KB | 运行时监控 |
| PSRAM 使用 | < 512KB | heap_caps_print_heap_info |
| 模型内存 | ~21KB | model_loader_get_total_memory |

**测试脚本**:
```c
void test_memory_usage(void) {
    printf("=== 内存占用报告 ===\n");
    printf("内部 RAM: %zu / %zu 字节\n", 
           heap_caps_get_total_size(MALLOC_CAP_INTERNAL) - 
           heap_caps_get_free_size(MALLOC_CAP_INTERNAL),
           heap_caps_get_total_size(MALLOC_CAP_INTERNAL));
    printf("PSRAM: %zu / %zu 字节\n",
           heap_caps_get_total_size(MALLOC_CAP_SPIRAM) - 
           heap_caps_get_free_size(MALLOC_CAP_SPIRAM),
           heap_caps_get_total_size(MALLOC_CAP_SPIRAM));
    printf("模型占用: %zu 字节\n", model_loader_get_total_memory());
}
```
