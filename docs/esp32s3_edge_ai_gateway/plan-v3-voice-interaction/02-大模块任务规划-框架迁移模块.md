## 技术要点

### ESP-IDF vs Zephyr API 对照

| 功能 | Zephyr API | ESP-IDF API |
|------|-----------|-------------|
| GPIO | `gpio_pin_configure()` | `gpio_set_direction()` |
| PWM | `pwm_set_cycles()` | `ledc_set_duty()` |
| UART | `uart_poll_out()` | `uart_write_bytes()` |
| I2S | `i2s_write()` | `i2s_channel_write()` |
| Thread | `k_thread_create()` | `xTaskCreate()` |
| Mutex | `k_mutex_lock()` | `xSemaphoreTake()` |
| Timer | `k_timer_start()` | `esp_timer_start()` |

## 风险与缓解

| 风险 | 概率 | 影响 | 缓解措施 |
|------|------|------|---------|
| ESP-IDF 版本兼容性 | 中 | 中 | 使用 ESP-IDF v5.4 LTS |
| 外设驱动差异 | 中 | 低 | 参考 ESP-IDF 示例 |
| 构建系统配置 | 低 | 低 | 使用标准模板 |
| **I2S 时钟冲突** | 中 | 高 | DAC 和麦克风使用独立 I2S 通道 |
