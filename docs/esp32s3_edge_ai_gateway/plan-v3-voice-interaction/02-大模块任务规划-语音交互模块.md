## 技术要点

### ESP-SR 架构

```
┌─────────────────────────────────────────────────────────────┐
│                      ESP-SR 架构                             │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  ┌──────────┐    ┌──────────┐    ┌──────────┐             │
│  │ 麦克风    │───→│  AFE     │───→│ WakeNet  │ 唤醒检测    │
│  │ I2S      │    │ 音频前端  │    │ 唤醒词    │             │
│  └──────────┘    └──────────┘    └────┬─────┘             │
│                                        │                    │
│                                        ↓ 唤醒后             │
│                                   ┌──────────┐             │
│                                   │ MultiNet │ 命令识别    │
│                                   │ 命令词    │             │
│                                   └────┬─────┘             │
│                                        │                    │
│                                        ↓                    │
│                                   ┌──────────┐             │
│                                   │ 命令处理  │             │
│                                   │ Handler  │             │
│                                   └──────────┘             │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### 命令词分类

| 类型 | 数量 | 示例 | 动作类型 |
|------|------|------|---------|
| 场景切换 | 8 | "打开专注模式" | 绝对值设置 |
| 亮度调节 | 6 | "调亮一点" | 相对调节 |
| 色温调节 | 6 | "换成暖光" | 绝对值设置 |
| 模式切换 | 4 | "切换演示模式" | 状态切换 |
| 状态查询 | 5 | "现在是什么模式" | TTS 回复 |
| 取消操作 | 3 | "恢复自动" | 状态恢复 |

### 关键配置参数

```c
// sdkconfig 关键配置
CONFIG_ESP_SR_WAKENET_MODEL="wn_xiaobaitong"
CONFIG_ESP_SR_WAKENET_THRESHOLD=0.5
CONFIG_ESP_SR_MULTINET_MODEL="mn_chinese"
CONFIG_ESP_SR_AFE_AEC_ENABLE=y
CONFIG_ESP_SR_AFE_NS_ENABLE=y
CONFIG_ESP_SR_AFE_BSS_ENABLE=n  // 单麦克风不需要
```

## 验收标准

| 检查项 | 验收标准 |
|--------|---------|
| 唤醒词识别率 | > 95%（安静环境） |
| 命令词识别率 | > 90%（安静环境） |
| 唤醒延迟 | < 500ms |
| 命令识别延迟 | < 300ms |
| 误唤醒率 | < 1 次/小时 |
| 内存占用 | < 600KB（语音模块） |

---

## 风险与缓解

| 风险 | 概率 | 影响 | 缓解措施 |
|------|------|------|---------|
| 唤醒词识别率低 | 中 | 高 | 优化 AFE 参数、增加训练数据 |
| 命令词混淆 | 中 | 中 | 增加命令词区分度 |
| 内存不足 | 中 | 高 | 使用 PSRAM、分步加载 |
| 噪声环境性能下降 | 高 | 中 | 优化 NS 参数、增加 VAD |
