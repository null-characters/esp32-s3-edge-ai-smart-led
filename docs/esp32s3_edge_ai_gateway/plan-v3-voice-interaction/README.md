## 核心技术决策

本规划基于以下架构改进：

1. **DAC + 扬声器**：MAX98357A I2S DAC，解决 TTS 输出问题
2. **AEC 回声消除**：扬声器与麦克风共存时的必需配置
3. **AFE 独占架构**：RingBuffer 消费者模型，解决麦克风资源竞争
4. **TTL 租约机制**：语音锁自动释放，防止死锁

详见 `01-语音交互升级总体规划.md`。
