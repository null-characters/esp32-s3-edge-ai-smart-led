## 技术要点

### TFLM 内存配置

```c
// tflm_allocator.c
#define TFLM_ARENA_SIZE (100 * 1024)  // 100KB

static uint8_t *tflm_arena = NULL;

esp_err_t tflm_init(void) {
    // 使用 PSRAM 分配大块内存
    tflm_arena = (uint8_t *)heap_caps_malloc(
        TFLM_ARENA_SIZE, 
        MALLOC_CAP_SPIRAM | MALLOC_CAP_8BIT
    );
    
    if (tflm_arena == NULL) {
        ESP_LOGE(TAG, "Failed to allocate TFLM arena");
        return ESP_ERR_NO_MEM;
    }
    
    return ESP_OK;
}
```

### Zephyr → ESP-IDF API 对照

| 功能 | Zephyr | ESP-IDF |
|------|--------|---------|
| 线程创建 | `k_thread_create()` | `xTaskCreatePinnedToCore()` |
| 互斥锁 | `k_mutex` | `SemaphoreHandle_t` |
| 消息队列 | `k_msgq` | `xQueue` |
| 定时器 | `k_timer` | `esp_timer` |
| 日志 | `LOG_INF()` | `ESP_LOGI()` |
| 延时 | `k_msleep()` | `vTaskDelay()` |
