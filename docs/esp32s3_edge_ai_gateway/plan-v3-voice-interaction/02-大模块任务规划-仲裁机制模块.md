## 技术要点

### 优先级策略

```
优先级顺序：语音命令 > 自动决策 > 默认规则

┌─────────────────────────────────────────────────────────┐
│                    仲裁流程                              │
├─────────────────────────────────────────────────────────┤
│                                                         │
│  语音命令 ──→ 有效？ ──→ Yes ──→ 执行语音命令           │
│              │                                          │
│              No                                         │
│              │                                          │
│              ↓                                          │
│  自动决策 ──→ 有效？ ──→ Yes ──→ 执行自动决策           │
│              │                                          │
│              No                                         │
│              │                                          │
│              ↓                                          │
│  默认规则 ──→ 执行默认规则                              │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

### 状态机设计

```
状态转换图：

           ┌──────────────┐
           │   AUTO_MODE   │ ←── 初始状态
           │   自动控制    │
           └───────┬──────┘
                   │
        语音命令   │
                   ↓
           ┌──────────────┐
           │  MANUAL_MODE │
           │   手动控制    │
           └───────┬──────┘
                   │
    超时/恢复命令  │
                   ↓
           ┌──────────────┐
           │   AUTO_MODE   │
           └──────────────┘
```
