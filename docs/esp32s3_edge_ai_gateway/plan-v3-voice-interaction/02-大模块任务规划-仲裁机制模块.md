## 技术要点

### 优先级策略

```
优先级顺序：语音命令 > 自动决策 > 默认规则

┌─────────────────────────────────────────────────────────┐
│                    仲裁流程                              │
├─────────────────────────────────────────────────────────┤
│                                                         │
│  语音命令 ──→ 有效？ ──→ Yes ──→ 执行语音命令           │
│              │            └── 启动 TTL 租约             │
│              No                                         │
│              │                                          │
│              ↓                                          │
│  自动决策 ──→ 有效？ ──→ Yes ──→ 执行自动决策           │
│              │                                          │
│              No                                         │
│              │                                          │
│              ↓                                          │
│  默认规则 ──→ 执行默认规则                              │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

### TTL 租约状态机 ⭐ 防止语音锁死

```
状态转换图：

           ┌──────────────┐
           │   AUTO_MODE   │ ←── 初始状态
           │   自动控制    │
           └───────┬──────┘
                   │
        语音命令   │
                   ↓
           ┌──────────────┐
           │  MANUAL_MODE │
           │   手动控制    │ ←── 启动 TTL 租约（默认 2 小时）
           └───────┬──────┘
                   │
    ┌──────────────┼──────────────┐
    │              │              │
    ▼              ▼              ▼
 TTL 过期    环境状态变化   用户说"恢复自动"
 (时间到)   (人离开>10分钟)
    │              │              │
    └──────────────┴──────────────┘
                   │
                   ↓
           ┌──────────────┐
           │   AUTO_MODE   │ ←── 自动释放语音锁
           └──────────────┘
```

### 统一事件格式

```c
typedef struct {
    uint8_t source;      // 事件来源：VOICE/AUTO/DEFAULT
    uint8_t type;        // 事件类型：SCENE/BRIGHT/COLOR
    uint16_t ttl_ms;     // TTL 租约时长（仅语音事件有效）
    union {
        scene_config_t scene;
        bright_config_t bright;
        color_config_t color;
    } data;
} Lighting_Event_t;
```

## 验收标准

| 检查项 | 验收标准 |
|--------|---------|
| 语音优先 | 语音命令立即生效 |
| 自动兜底 | 无语音时自动决策 |
| TTL 租约 | 默认 2 小时后自动释放语音锁 |
| 环境退出 | 人离开 > 10 分钟自动释放 |
| 手动恢复 | 用户说"恢复自动"立即释放 |
| 状态持久化 | 重启后恢复状态 |

## 风险与缓解

| 风险 | 概率 | 影响 | 缓解措施 |
|------|------|------|---------|
| 语音锁死 | 中 | 高 | TTL 租约 + 环境变化检测自动释放 |
| 状态不同步 | 中 | 中 | 统一事件队列消费 |
| TTL 配置不合理 | 低 | 中 | 可配置 TTL 时长 |
