## 核心技术决策

本规划基于以下架构改进：

1. **双层思维模型**：本地 L1（快速响应）+ 云端 L2（深度理解）
2. **意图分流路由器**：90% 本地执行，10% 上云
3. **五维记忆系统**：Profile/Preference/Fact/Rule/Context
4. **Lua 动态规则**：云端下发脚本，本地执行
5. **优雅降级**：断网时回退 v3 本地模式

详见 `01-云端智能体升级总体规划.md`。
