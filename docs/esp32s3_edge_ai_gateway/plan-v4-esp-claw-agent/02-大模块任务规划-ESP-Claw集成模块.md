## 技术要点

### ESP-Claw 架构

```
┌─────────────────────────────────────────────────────────────┐
│                      ESP-Claw 架构                             │
├─────────────────────────────────────────────────────────────┤
│  ┌──────────┐    ┌──────────┐    ┌──────────┐              │
│  │ 语音输入  │───→│  Agent   │───→│ 云端API  │              │
│  │ (唤醒后)  │    │  主循环   │    │ DeepSeek │              │
│  └──────────┘    └──────────┘    └────┬─────┘              │
│                                       │                     │
│                                       ▼                     │
│                               ┌──────────────┐             │
│                               │  五维记忆    │             │
│                               │  系统管理    │             │
│                               └────┬────────┘             │
│                                    │                      │
│                                    ▼                      │
│                               ┌──────────────┐             │
│                               │  工具调用    │             │
│                               │  执行结果    │             │
│                               └──────────────┘             │
└─────────────────────────────────────────────────────────────┘
```

// api_client.c 核心配置
