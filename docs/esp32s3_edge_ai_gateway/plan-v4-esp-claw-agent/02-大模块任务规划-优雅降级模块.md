## 技术要点

### 网络状态映射

| 网络状态 | 唤醒灯颜色 | 运行模式 | 语音提示 |
|---------|-----------|---------|---------|
| 网络畅通 | 蓝色 | Agent 模式 | 无 |
| Wi-Fi 断开 | 绿色 | v3 本地模式 | "网络未连接，已切换至本地控制"（仅一次） |
| API 超时 (>3s) | 绿色 | v3 本地模式 | 同上 |

### 降级状态机

```
AGENT_MODE (云端+本地)
      │
      │ 网络断开
      ▼
LOCAL_MODE (v3本地)
      │
      │ 用户语音唤醒
      ▼
提示网络断开（仅一次）
      │
      │ 网络恢复
      ▼
AGENT_MODE
```

---

## UX 原则

1. **语音提示极度克制**：绝不允许"网络异常请重试"死循环
2. **仅提示一次**：降级后首次唤醒提示，后续不再重复
3. **功能不丢失**：降级后保留所有 v3 功能

---

## 验收标准

| 检查项 | 验收标准 |
|--------|---------|
| 降级响应时间 | < 3s |
| 提示次数 | 仅一次 |
| v3 功能保留 | 100% |
| 恢复响应时间 | < 5s |
