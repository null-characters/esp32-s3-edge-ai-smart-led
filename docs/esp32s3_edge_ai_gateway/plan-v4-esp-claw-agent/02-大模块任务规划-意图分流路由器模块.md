## 技术要点

### 分流决策流程

```
语音唤醒 → MultiNet识别
              │
    ┌─────────┴─────────┐
    │                   │
    ▼                   ▼
命中本地指令        未命中
    │                   │
    ▼                   ▼
本地执行          录音上传云端
<200ms            1-3s
```
