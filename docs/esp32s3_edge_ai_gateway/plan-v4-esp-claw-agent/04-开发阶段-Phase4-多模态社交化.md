**降级条件与恢复：**

| 条件类型 | 触发条件 | 动作 |
|---------|---------|------|
| Wi-Fi 断开 | 连续 5s 无连接 | 进入降级模式 |
| API 超时 | 连续 3 次超时 | 进入降级模式 |
| DNS 失败 | 连续 3 次失败 | 进入降级模式 |
| 网络恢复 | Wi-Fi 重连成功 | 退出降级模式 |
| API 恢复 | 连续 2 次成功 | 退出降级模式 |

## 验收标准

| 检查项 | 标准 | 测试方法 |
|--------|------|---------|
| 消息响应延迟 | <2s | 端到端计时 |
| 指令执行成功率 | >95% | 100 次指令测试 |
| 降级切换时间 | <1s | 断网测试 |
| 恢复同步时间 | <5s | 重连测试 |
| LED 状态正确 | 100% | 状态机测试 |
