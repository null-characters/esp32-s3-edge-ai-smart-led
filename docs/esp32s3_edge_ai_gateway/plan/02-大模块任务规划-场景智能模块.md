# 大模块任务规划 - 场景智能模块

## 模块概述

| 属性 | 说明 |
|------|------|
| 模块名称 | 场景智能模块 |
| 所属阶段 | Phase 3 |
| 计划周期 | 2周 |
| 核心目标 | 基于多模态识别，实现场景级智能控制，平滑过渡 |

## 功能边界

### 包含功能
- 场景规则设计与映射
- 真实场景数据采集
- 模型优化与精度提升
- 场景平滑过渡算法
- 场景状态机管理

### 不包含功能
- 雷达/麦克风驱动（Phase 1/2 已完成）
- 模型训练框架（Phase 2 已完成）
- 端侧学习（Phase 4）

## 关键数据结构

### 场景定义
```c
typedef enum {
    SCENE_MEETING = 0,      // 会议讨论
    SCENE_FOCUS = 1,        // 专注工作
    SCENE_PRESENTATION = 2, // 汇报演示
    SCENE_RELAX = 3,        // 休息放松
    SCENE_CLEANING = 4,     // 打扫卫生
    SCENE_UNKNOWN = 255     // 未知场景
} scene_id_t;

typedef struct {
    scene_id_t id;
    const char *name;
    uint16_t target_color_temp;  // 目标色温
    uint8_t target_brightness;   // 目标亮度
    uint8_t transition_sec;      // 过渡时间
} scene_definition_t;
```

### 场景状态
```c
typedef struct {
    scene_id_t current_scene;
    scene_id_t previous_scene;
    uint32_t scene_start_time;
    uint32_t scene_duration;
    float confidence;
    uint8_t stability_counter;  // 稳定性计数
} scene_state_t;
```

### 平滑过渡
```c
typedef struct {
    uint16_t start_ct;
    uint16_t target_ct;
    uint8_t start_br;
    uint8_t target_br;
    uint32_t start_time;
    uint32_t duration_ms;
    bool active;
} smooth_transition_t;
```
