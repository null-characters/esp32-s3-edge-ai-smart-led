# 大模块任务规划 - 雷达接入模块

## 模块概述

| 属性 | 说明 |
|------|------|
| 模块名称 | 雷达接入模块 |
| 所属阶段 | Phase 1 |
| 计划周期 | 2周 |
| 核心目标 | 引入毫米波雷达 LD2410B，替换 PIR，实现真 AI 场景判断 |

## 功能边界

### 包含功能
- LD2410B 硬件集成与驱动开发
- 雷达数据采集与解析（距离、速度、能量、目标数）
- 8 维雷达特征工程
- 雷达分析器 MLP 模型训练与量化
- TFLM 端侧推理集成
- 场景测试验证

### 不包含功能
- 麦克风音频采集（Phase 2）
- 多模态融合（Phase 2）
- 场景智能控制（Phase 3）
- 端侧学习（Phase 4）

## 依赖关系

```
plan-v1 (PIR + MLP)
       │
       ▼
Phase 1 (雷达接入模块)
       │
       ├── 复用: UART通信框架、TFLM框架
       └── 替换: PIR驱动 → LD2410B驱动
              MLP 7→16→2 → 雷达MLP 8→16→6
```

## 关键数据结构

### LD2410B 原始数据
```c
typedef struct {
    float distance;          // 目标距离 0-8m
    float velocity;          // 目标速度 -5~+5 m/s
    float energy;            // 目标能量
    uint8_t target_count;    // 目标数量
    uint8_t motion_state;    // 0=无, 1=静止, 2=运动
} ld2410_data_t;
```

### 雷达特征向量
```c
typedef struct {
    float distance;           // 目标距离归一化 [0,1]
    float velocity;           // 目标速度归一化 [0,1]
    float energy;             // 目标能量归一化 [0,1]
    float target_count;       // 目标数量归一化 [0,1]
    float motion_amplitude;   // 运动幅度 [0,1]
    float motion_period;      // 运动周期 [0,1]
    float distance_variance;  // 距离方差 [0,1]
    float energy_trend;       // 能量趋势 [-1,1]
} radar_features_t;  // 8 维
```

### 雷达分析器输出
```c
typedef struct {
    float features[6];        // 6 维特征向量
    uint8_t scene_hint;       // 场景提示 (可选)
} radar_analyzer_output_t;
```

## 关键接口定义

### LD2410B 驱动接口
```c
int ld2410_init(void);
int ld2410_read(ld2410_data_t *data);
int ld2410_send_cmd(uint8_t cmd, const uint8_t *data, uint8_t len);
```

### 特征工程接口
```c
void radar_extract_features(const ld2410_data_t *raw, radar_features_t *feat);
float detect_motion_period(const ld2410_data_t *history, int count);
float calculate_distance_variance(const ld2410_data_t *history, int count);
```

### 推理接口
```c
int radar_analyzer_init(void);
int radar_analyzer_infer(const radar_features_t *input, 
                         radar_analyzer_output_t *output);
```

## 模型参数

| 参数 | 值 |
|------|-----|
| 网络结构 | MLP: 8→16→12→6 |
| 输入维度 | 8 |
| 隐藏层1 | 16 神经元, ReLU |
| 隐藏层2 | 12 神经元, ReLU |
| 输出层 | 6 神经元, Linear |
| 参数量 | ~400 |
| 模型大小 | ~2KB (INT8) |
| 推理延迟 | <10ms |

## 里程碑

- **M1** (Week 1 中): LD2410B 驱动完成，数据正确解析
- **M2** (Week 2 末): 雷达分析器推理正确，场景测试通过

## 与 plan-v1 的衔接

```
plan-v1 提供:
  - UART 通信框架 (可复用)
  - TFLM 推理框架 (可复用)
  - 后处理逻辑 (可复用)

Phase 1 替换:
  - PIR 驱动 → LD2410B 驱动
  - GPIO 输入 → UART1 输入
  - 1bit 状态 → 8 维特征
  - MLP 7→16→2 → MLP 8→16→12→6
```

## 验收场景

| 场景 | 雷达特征 | 预期输出 |
|------|----------|----------|
| 人静坐呼吸 | 距离固定，微动周期 ~0.3Hz | 静止人存在 |
| 风扇摇头 | 周期运动，距离变化 | 非人运动 |
| 多人站立 | 多目标，距离分散 | 多人场景 |
| 人走动 | 距离变化，能量高 | 运动人 |
| 无人 | 无目标 | 无人状态 |
