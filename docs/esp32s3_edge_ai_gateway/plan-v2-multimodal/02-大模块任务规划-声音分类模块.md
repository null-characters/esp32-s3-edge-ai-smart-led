# 大模块任务规划 - 声音分类模块

## 模块概述

| 属性 | 说明 |
|------|------|
| 模块名称 | 声音分类模块 |
| 所属阶段 | Phase 2 |
| 计划周期 | 3周 |
| 核心目标 | 引入 MEMS 麦克风 INMP441，实现声音场景分类，双模态融合 |

## 功能边界

### 包含功能
- INMP441 硬件集成与 I2S 驱动开发
- 音频数据采集与缓冲管理
- MFCC 特征提取
- 声音分类 CNN 模型训练与量化
- 双模态融合模型训练
- 多模型推理调度

### 不包含功能
- 雷达驱动开发（Phase 1 已完成）
- 场景智能控制（Phase 3）
- 端侧学习（Phase 4）

## 关键数据结构

### 音频缓冲区
```c
#define AUDIO_SAMPLE_RATE   16000
#define AUDIO_BUFFER_SIZE   1024
#define AUDIO_BUFFER_COUNT  4

typedef struct {
    int16_t buffer[AUDIO_BUFFER_COUNT][AUDIO_BUFFER_SIZE];
    volatile uint8_t write_idx;
    volatile uint8_t read_idx;
    k_mutex_t lock;
} audio_ring_buffer_t;
```

### MFCC 特征
```c
#define MFCC_NUM        40      // 40 个 MFCC 系数
#define MFCC_FRAMES     32      // 32 帧

typedef struct {
    float data[MFCC_NUM][MFCC_FRAMES];  // 40 x 32 特征矩阵
} mfcc_features_t;
```

### 声音分类输出
```c
typedef enum {
    SOUND_SILENCE = 0,      // 寂静
    SOUND_SINGLE_VOICE,     // 单人讲话
    SOUND_MULTI_VOICE,      // 多人讨论
    SOUND_KEYBOARD,         // 键盘敲击
    SOUND_OTHER             // 其他噪音
} sound_class_t;

typedef struct {
    float probs[5];         // 5 类概率
    sound_class_t top_class;
    float confidence;
} sound_classifier_output_t;
```

### 多模态融合输入
```c
typedef struct {
    float sound_probs[5];     // 声音分类概率
    float radar_features[6];  // 雷达分析器输出
    float env_features[8];    // 环境特征 (时间/天气/日落)
} multimodal_input_t;
```

### 多模态融合输出
```c
typedef struct {
    uint16_t color_temp;      // 色温 2700-6500K
    uint8_t brightness;       // 亮度 0-100%
    uint8_t scene_id;         // 识别的场景 ID
    float confidence;         // 置信度
} multimodal_output_t;
```

## 关键接口定义

### I2S 驱动接口
```c
int inmp441_init(void);
int inmp441_read(int16_t *buffer, size_t samples, k_timeout_t timeout);
bool inmp441_is_data_ready(void);
```

### MFCC 提取接口
```c
int mfcc_init(void);
void mfcc_extract(const int16_t *audio, size_t samples, mfcc_features_t *out);
```

### 声音分类接口
```c
int sound_classifier_init(void);
int sound_classifier_infer(const mfcc_features_t *input, 
                           sound_classifier_output_t *output);
```

### 多模态融合接口
```c
int multimodal_init(void);
int multimodal_infer(const multimodal_input_t *input, 
                     multimodal_output_t *output);
```

## 模型参数

### 声音分类 CNN
| 参数 | 值 |
|------|-----|
| 输入形状 | (40, 32, 1) MFCC |
| Conv2D 1 | 16 filters, 3x3, ReLU |
| MaxPool 1 | 2x2 |
| Conv2D 2 | 32 filters, 3x3, ReLU |
| MaxPool 2 | 2x2 |
| Flatten | - |
| Dense 1 | 32, ReLU |
| Dense 2 | 5, Softmax |
| 模型大小 | ~30KB (INT8) |
| 推理延迟 | ~50ms |

### 多模态融合 MLP
| 参数 | 值 |
|------|-----|
| 输入维度 | 19 (5+6+8) |
| Dense 1 | 32, ReLU |
| Dense 2 | 16, ReLU |
| Dense 3 | 2, Sigmoid |
| 模型大小 | ~3KB (INT8) |
| 推理延迟 | ~5ms |
