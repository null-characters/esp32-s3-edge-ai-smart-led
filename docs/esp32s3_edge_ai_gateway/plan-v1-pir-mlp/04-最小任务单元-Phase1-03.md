# 最小任务单元 - Phase 1 (03/03)

> **Phase**: 基础硬件通信模块  
> **范围**: 任务 HW-021 ~ HW-023  
> **预计工时**: 2天

---

## 任务 HW-021: main.c集成

**目标**: 将各模块集成到主程序

**执行步骤**:
1. 创建 `src/main.c`
2. 包含所有模块头文件
3. 实现 `main()` 初始化流程
4. 实现主控制循环
5. 添加错误处理和日志

**验收标准**:
- 程序可编译通过
- 主循环运行稳定

---

## 任务 HW-022: 构建测试

**目标**: 验证项目构建流程

**执行步骤**:
1. 执行 `west build -b esp32s3_devkitc`
2. 检查编译输出无错误
3. 确认固件大小合理
4. 记录构建日志

**验收标准**:
- 构建成功
- 无编译警告

---

## 任务 HW-023: 功能验证

**目标**: 完整功能测试

**执行步骤**:
1. 烧录固件到ESP32-S3
2. 测试人员进入→灯亮
3. 测试人员离开→5分钟后灯灭
4. 测试时段切换→色温光强变化
5. 记录测试结果

**验收标准**:
- 自动开关灯正常
- 时段切换正常
