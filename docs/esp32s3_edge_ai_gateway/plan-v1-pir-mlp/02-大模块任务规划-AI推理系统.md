# 大模块任务规划 - AI推理系统模块

## 模块概述

| 属性 | 说明 |
|------|------|
| 模块名称 | AI推理系统模块 |
| 所属阶段 | Phase 3 |
| 计划周期 | 2周 |
| 核心目标 | 集成TFLite Micro，实现神经网络智能调光 |

## 功能边界

### 包含功能
- PC端模型训练与量化
- TFLite Micro集成
- 7维特征工程
- 神经网络推理
- 数据记录与缓存
- 后处理与渐变控制

### 不包含功能
- WiFi数据获取（由Phase 2提供）
- UART设备控制（由Phase 1提供）
- 在线学习（后续迭代）
- 手机App交互

## 关键数据结构

### 特征向量
```c
struct lighting_features {
    float hour_sin;           // 时间正弦编码
    float hour_cos;           // 时间余弦编码
    float sunset_proximity;   // 日落临近度 [-1,1]
    float sunrise_hour_norm;  // 日出时间归一化 [0,1]
    float sunset_hour_norm;   // 日落时间归一化 [0,1]
    float weather;            // 天气编码 [0,1]
    float presence;           // 人员存在 {0,1}
};
```

### 照明事件记录
```c
struct lighting_event {
    uint32_t timestamp;       // Unix时间戳
    uint16_t color_temp;      // 色温(K)
    uint8_t brightness;       // 光强(%)
    float weather;            // 天气编码
    uint8_t presence;         // 人员存在
    uint8_t source;           // 来源(0=AI, 1=规则)
};
```

## 关键接口定义

### TFLM接口
```c
int ai_init(void);
void ai_infer(const struct lighting_features *features,
              uint16_t *out_color_temp, uint8_t *out_brightness);
```

### 特征工程接口
```c
void feature_extract(struct lighting_features *out);
float calculate_hour_sin(float hour);
float calculate_hour_cos(float hour);
float calculate_sunset_proximity(float hour, float sunset_hour);
```

### 后处理接口
```c
void ai_post_process(uint16_t *color_temp, uint8_t *brightness, 
                     bool presence, int hour);
void smooth_transition(uint16_t target_ct, uint8_t target_br, 
                       uint8_t duration_sec);
```

### 数据记录接口
```c
int event_record_init(void);
int event_log(const struct lighting_event *event);
int event_query(time_t start, time_t end, struct lighting_event *out, 
                int max_count);
```

## 模型参数

| 参数 | 值 |
|------|-----|
| 网络结构 | MLP: 7→32→16→2 |
| 输入维度 | 7 |
| 隐藏层1 | 32神经元, ReLU |
| 隐藏层2 | 16神经元, ReLU |
| 输出层 | 2神经元, Linear |
| 参数量 | ~1.2K |
| 模型大小 | ~4KB (INT8) |
| 推理延迟 | <5ms |

## 里程碑

- **M5** (Week 5中): PC端模型训练完成，转换为C数组
- **M6** (Week 6末): AI推理驱动调光，输出平滑连续，系统稳定运行

## 与Phase 1/2的衔接

```
Phase 1提供: uart_send_set_both(ct, br)  // 设备控制
Phase 2提供: weather_cache, sun_cache    // 环境数据
                  │
                  ▼
Phase 3 AI推理流程:
  1. 获取环境数据 (weather, sunrise, sunset)
  2. 读取PIR状态 (presence)
  3. 计算7维特征 (feature_extract)
  4. TFLM推理 (ai_infer)
  5. 后处理 (无人关灯、深夜限幅)
  6. 渐变输出 (uart_send_set_both)
  7. 记录事件 (event_log)
```
