# 最小任务单元 - Phase 2 (04/04)

> **Phase**: WiFi数据服务模块  
> **范围**: 任务 WF-031 ~ WF-034  
> **预计工时**: 2天

---

## 任务 WF-031: WiFi管理线程

**目标**: 创建独立WiFi管理任务

**执行步骤**:
1. 创建WiFi线程
2. 设置线程优先级
3. 实现状态机管理
4. 处理连接/断开事件
5. 与主线程通信

---

## 任务 WF-032: 主循环集成

**目标**: 集成WiFi服务到主循环

**执行步骤**:
1. 在主程序初始化WiFi
2. 启动WiFi线程
3. 等待WiFi就绪
4. 获取时间/天气/日落数据
5. 更新调光参数

---

## 任务 WF-033: 离线处理

**目标**: 实现离线默认配置

**执行步骤**:
1. 检测WiFi断开
2. 使用默认天气值
3. 使用默认日落时间
4. 继续基础调光功能
5. 记录离线状态

---

## 任务 WF-034: 功能测试

**目标**: Phase 2完整功能测试

**执行步骤**:
1. WiFi连接测试
2. 时间同步测试
3. 天气获取测试
4. 日落获取测试
5. 综合场景测试
