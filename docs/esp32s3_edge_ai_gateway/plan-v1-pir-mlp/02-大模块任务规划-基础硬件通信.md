# 大模块任务规划 - 基础硬件通信模块

## 模块概述

| 属性 | 说明 |
|------|------|
| 模块名称 | 基础硬件通信模块 |
| 所属阶段 | Phase 1 |
| 计划周期 | 2周 |
| 核心目标 | 建立ESP32-S3与驱动板的UART通信，实现PIR传感器驱动，完成基础调光控制 |

## 功能边界

### 包含功能
- PIR人体传感器GPIO输入检测
- UART2与驱动板通信协议实现
- 基础规则驱动的调光逻辑
- 硬件初始化与配置

### 不包含功能
- WiFi网络连接
- 天气数据获取
- AI神经网络推理
- 手机App交互

## 关键接口定义

### GPIO PIR接口
```c
int pir_init(void);
bool pir_get_status(void);
bool pir_get_status_debounced(void);
```

### UART驱动接口
```c
int uart_driver_init(void);
int uart_send_set_color_temp(uint16_t color_temp);
int uart_send_set_brightness(uint8_t brightness);
int uart_send_set_both(uint16_t color_temp, uint8_t brightness);
int uart_send_get_status(void);
```

### 控制逻辑接口
```c
void rule_get_lighting_params(bool presence, float hour,
                              uint16_t *color_temp, uint8_t *brightness);
void fade_to_target(uint16_t target_color_temp, uint8_t target_brightness, 
                    uint8_t duration_sec);
```
