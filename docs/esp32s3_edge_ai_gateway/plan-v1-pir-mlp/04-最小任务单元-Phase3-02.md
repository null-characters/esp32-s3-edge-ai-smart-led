# 最小任务单元 - Phase 3 (02/03)

> **Phase**: AI推理系统模块  
> **范围**: 任务 AI-011 ~ AI-020  
> **预计工时**: 5天

---

## 任务 AI-011: 张量分配

**目标**: 分配输入输出张量

**执行步骤**:
1. 调用AllocateTensors()
2. 验证分配成功
3. 获取输入张量指针
4. 获取输出张量指针
5. 检查张量维度

---

## 任务 AI-012: 推理验证

**目标**: 验证推理功能

**执行步骤**:
1. 填充测试输入
2. 调用Invoke()执行推理
3. 读取输出张量
4. 对比预期结果
5. 记录推理时间

---

## 任务 AI-013: 特征结构定义

**目标**: 定义特征数据结构

**执行步骤**:
1. 创建 `ai_features.h`
2. 定义 `lighting_features_t` 结构体
3. 包含7个float字段
4. 添加特征有效性标志
5. 定义特征数量常量

---

## 任务 AI-014: 时间特征计算

**目标**: 计算时间正弦/余弦特征

**执行步骤**:
1. 获取当前小时
2. 计算hour_sin: sin(hour * 2π / 24)
3. 计算hour_cos: cos(hour * 2π / 24)
4. 填充到特征结构
5. 添加单元测试

---

## 任务 AI-015: 日落临近度

**目标**: 计算日落临近度特征

**执行步骤**:
1. 获取当前时间和日落时间
2. 计算时间差(小时)
3. 归一化到-1~1范围
4. 日落前为负值(接近0)
5. 填充到特征结构

---

## 任务 AI-016: 特征向量组装

**目标**: 组装完整特征向量

**执行步骤**:
1. 收集所有特征值
2. 归一化到0-1或-1~1
3. 填充7维数组
4. 验证无NaN/Inf
5. 返回特征结构

---

## 任务 AI-017: 推理接口封装

**目标**: 封装AI推理接口

**执行步骤**:
1. 创建 `ai_infer()` 函数
2. 接收特征向量参数
3. 填充输入张量
4. 执行推理
5. 返回色温和光强

---

## 任务 AI-018: 后处理逻辑

**目标**: 实现推理结果后处理

**执行步骤**:
1. 检查人员状态
2. 无人则强制关灯
3. 深夜限制最大亮度
4. 限制输出范围
5. 返回最终值

---

## 任务 AI-019: 渐变控制集成

**目标**: 集成渐变控制

**执行步骤**:
1. 获取AI输出值
2. 与当前值比较差值
3. 如差值大则启动渐变
4. 平滑过渡到目标值
5. 避免输出跳变

---

## 任务 AI-020: 推理线程

**目标**: 创建推理定时线程

**执行步骤**:
1. 创建推理线程
2. 设置30秒周期
3. 循环执行推理
4. 更新灯具状态
5. 添加停止机制
