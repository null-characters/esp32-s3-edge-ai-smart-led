# Zephyr + ESP32-S3 开发环境搭建报告

## 平台变更说明

**⚠️ 重要变更**: 本项目已正式从 **Windows** 平台迁移至 **macOS** 平台进行开发。

### 变更原因

| 问题 | Windows + WSL2 | macOS |
|------|----------------|-------|
| native_sim 单元测试 | ❌ WSL2 跨文件系统挂载性能极差，构建耗时 20 分钟+ | ✅ 原生支持，构建快速 |
| 文件系统性能 | ❌ `/mnt/e/` 挂载导致 IO 瓶颈 | ✅ 本地 APFS，性能优秀 |
| 工具链复杂度 | ❌ 需 Windows + WSL2 双重配置 | ✅ 单一环境，配置简洁 |
| 开发体验 | ⚠️ 跨系统切换繁琐 | ✅ Unix 原生体验一致 |

## macOS 环境搭建指南

### 1. 安装 Homebrew

```bash
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
```

### 2. 安装基础工具

```bash
brew install cmake ninja gperf python3 git qemu
```

### 3. 安装 west

```bash
pip3 install --user west
```

添加到 `~/.zshrc` 或 `~/.bash_profile`：
```bash
echo 'export PATH="$HOME/.local/bin:$PATH"' >> ~/.zshrc
source ~/.zshrc
```

### 4. 克隆项目仓库

```bash
# 选择你的工作目录
cd ~/workspace

# 克隆仓库（使用你实际的仓库地址）
git clone https://github.com/null-characters/esp32-s3-edge-ai-smart-led.git zephyrproject
cd zephyrproject
```

### 5. 初始化 west workspace

```bash
# 初始化 west，使用当前目录作为 workspace
west init -l

# 更新所有模块
west update
```

### 6. 安装 Zephyr Python 依赖

```bash
pip3 install --user -r zephyr/scripts/requirements.txt
```

### 7. 安装 Zephyr SDK

```bash
# 下载 SDK（版本 0.17.0）
cd ~
wget https://github.com/zephyrproject-rtos/sdk-ng/releases/download/v0.17.0/zephyr-sdk-0.17.0_macos-aarch64.tar.xz

# 解压到指定目录
tar xvf zephyr-sdk-0.17.0_macos-aarch64.tar.xz -C ~/zephyr-sdk

# 运行设置脚本
cd ~/zephyr-sdk/zephyr-sdk-0.17.0
./setup.sh

# 安装 ESP32 工具链
./setup.sh -t xtensa-espressif_esp32s3_zephyr-elf
```

### 8. 配置环境变量

编辑 `~/.zshrc`（或 `~/.bash_profile`）：

```bash
# Zephyr 环境变量
export ZEPHYR_BASE="$HOME/workspace/zephyrproject/zephyr"
export ZEPHYR_SDK_INSTALL_DIR="$HOME/zephyr-sdk/zephyr-sdk-0.17.0"
export PATH="$ZEPHYR_BASE/scripts:$PATH"
```

应用配置：
```bash
source ~/.zshrc
```

### 9. 注册 Zephyr 包

```bash
cd $ZEPHYR_BASE
west zephyr-export
```

### 10. 验证环境

编译 hello_world 示例：

```bash
cd ~/workspace/zephyrproject
west build -b esp32s3_devkitc/esp32s3/procpu zephyr/samples/hello_world -d build_hello
```

烧录到设备：
```bash
west flash -d build_hello
```

## 运行单元测试

Mac 原生支持 native_sim，单元测试构建快速：

```bash
cd ~/workspace/zephyrproject/zephyr

# 运行所有单元测试
./scripts/twister --testsuite-root ../prj/tests/unit -p native_sim

# 运行特定测试
./scripts/twister --testsuite-root ../prj/tests/unit -p native_sim -s prj.unit.command_parser
```

## 支持的 ESP32-S3 功能

- Dual-core Xtensa LX7 CPU (240MHz)
- Wi-Fi 4 (802.11 b/g/n)
- BLE 5.0 + BLE Mesh
- USB OTG
- 完整的外设支持 (I2C/SPI/UART/CAN/Touch/LCD/Camera)

## 注意事项

1. **首次设置**: 执行 `source ~/.zshrc` 或重新打开终端以加载环境变量

2. **开发板**: 使用 `esp32s3_devkitc/esp32s3/procpu` 作为目标板

3. **SDK 路径**: Mac 上 SDK 安装在 `~/zephyr-sdk/`，与 Windows 路径不同

4. **单元测试**: Mac 原生支持 native_sim，无需 WSL2，构建速度极快
